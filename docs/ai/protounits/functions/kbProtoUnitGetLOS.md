# kbProtoUnitGetLOS

## 1. Description

!!! note "Signature of `kbProtoUnitGetLOS`"

    ```cpp
    float kbProtoUnitGetLOS(int protoUnitID);
    ```

This function returns the vision range of the proto unit. The returned value is
in meters.
