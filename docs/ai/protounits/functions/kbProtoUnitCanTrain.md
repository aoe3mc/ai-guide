# kbProtoUnitCanTrain

## 1. Description

!!! note "Signature of `kbProtoUnitCanTrain`"

    ```cpp
    bool kbProtoUnitCanTrain(int sourceProtoUnitID, int targetProtoUnitID);
    ```

This function returns true if the source proto unit can train the target proto
unit.
