# kbProtoUnitAvailable

## 1. Description

!!! note "Signature of `kbProtoUnitAvailable`"

    ```cpp
    bool kbProtoUnitAvailable(int protoUnitID);
    ```

This function returns true if the proto unit is currently available, i.e.
if it can be trained or built.
